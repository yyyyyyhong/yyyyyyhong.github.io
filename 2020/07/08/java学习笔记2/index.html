java学习笔记2

发表于 2020-07-08

函数定义时用于传参的符号叫形式参数，简称形参，不需要在函数内部传递的参数叫实参

java类支持4中访问区分符

默认（friendly）: 同包可调用，子类不继承
public 同包可调用，子类继承
private 同包不可调用，子类不继承
protect 同包不可调用，子类可继承

创建类需要用到new运算符
如 Student zeng = new Student()
左边是声明一个类
右边用new运算符创建一个类
Student()是与类同名的成员函数（构造符函数）（可在类中自定义，也可以不定义使用系统的默认）

静态变量在类的对象创建前就可以通过类名进行访问

这里不小心用ctrl+Z退出了vi编辑器 查找后发现fg加数字可以恢复

静态函数只能访问静态变量和自身定义的变量

使用逻辑表达式时注意最好不要在逻辑表达式中进行数值计算过程
因为：有时逻辑表达式进行完左边的运算后就返回值了不再进行右边的运算，若在右边有运算则会被直接跳过出现短路的问题

java定义的常量会被java虚拟机优化
相同的常量会指向相同的内存
因此虽然常量间的 == 运算是指内存是否相等，
但是相同的常量经过 == 运算还是会返回True
